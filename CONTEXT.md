# LaunchLens Research Studio

LaunchLens Research Studio turns product ideas into evidence-backed research reports. This glossary keeps product-facing language consistent across report generation, report viewing, and downstream handoff.

## Language

**Output Profile**:
A reader-facing report mode that changes information density and explanation level while preserving the underlying evidence trail.
_Avoid_: User level, difficulty mode, beginner/pro mode

**Idea Profile**:
An Output Profile for individual builders who need plain-language validation, major risks, and one concrete next step.
_Avoid_: Basic mode, low-level mode

**Founder Profile**:
An Output Profile for early teams that need enough evidence to prioritize, execute, and hand off into go-to-market work.
_Avoid_: Default mode, startup mode

**Analyst Profile**:
An Output Profile for expert readers who need scores, citations, source trails, and raw supporting detail.
_Avoid_: Advanced mode, expert-only mode

**Analysis Companion**:
A contextual explanation panel that teaches how to read a report term, metric, or evidence marker without interrupting the report flow.
_Avoid_: Tooltip, help popup, tutorial
